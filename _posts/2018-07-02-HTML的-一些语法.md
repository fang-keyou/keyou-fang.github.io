### No.1 一些必要的东西

正如C++一样，什么主函数是必须的，HTML也不例外。

在html 5中，必须要在第一行打上这个：

```html
<!DOCTYPE html>
```
最简单的html程序为：

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>文档标题</title>
</head>
<body>
文档内容......
</body>
</html>
```
